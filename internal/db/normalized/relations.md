Таблица User - хранит такие данные пользователя, как имя пользователя, электронная почта и пароль.
Board - хранит такие данные досок, как id пользователя, которому принадлежит доска, название доски, ссылка на картинку, является ли доска активной или перемещена в архив,
время когда доска была создана и область видимости доски, которая моежт быть 3 видов: public - все кому была отправленна ссылка на эту доску, может её редактировать, team - только добавленные участники
могут редактировать доску, private - никто кроме администратора не может редактировать доску.
List - хранит список задач, объединёные по какому-то признаку (к примеру, все задачи находящиеся в работе). Имеет поля: id доски, которой принадлежит данный список. название списка,
дата создания списка, position- ?
board_member - служебная таблица для связи многие ко многим. Служит для связи досок с их участниками и хранить роль каждого участника. 
Card - таблица хранящая данные карточки с задачей. Эта таблица хранит id автора этой карточки, id списка в котором эта карточка состоит, position - ?, 
время создания карточки и (необязательно) время дедлайна  по задаче
Card_member - служебная таблица для связи многие ко многим между карточкой с задачей и участноками, занимающиеся этой задачей
Attachment - таблица хранящая id карты, к которому принадлежит это вложение, время создания вложения, ссылка на файл.
